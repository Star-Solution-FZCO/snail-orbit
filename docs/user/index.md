Welcome to the Snail Orbit user documentation. This section contains guides and references to help you use Snail Orbit effectively.

## Getting Started

Snail Orbit is a project management tool that combines issue tracking with agile boards and workflow automation. Here are some key concepts to help you get started:

- **Issues** - The core entities representing tasks, bugs, features, or any work items
- **Projects** - Containers that organize issues with role-based permissions
- **Custom Fields** - Extensible field system to capture project-specific data
- **Agile Boards** - Kanban-style boards with configurable columns and swimlanes
- **Workflows** - Python-based automation for issue lifecycle management
